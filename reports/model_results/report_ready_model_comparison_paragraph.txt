The improved Logistic Regression and ANN models were trained using SMOTE-based class balancing, cross-validated hyperparameter tuning, and threshold optimization. Improved ANN achieved the highest accuracy (0.9783), while Improved ANN produced the highest precision (0.3134). For fraud detection sensitivity, Improved ANN achieved the highest recall (0.2500), and Improved ANN achieved the highest F1-score (0.2781). Based on ROC-AUC, Improved ANN showed the strongest overall class discrimination ability with a score of 0.7346. These results suggest that the improved training pipeline provides better handling of the imbalanced dataset than a simple baseline model, especially when threshold tuning is used instead of relying only on the default 0.50 cutoff.
